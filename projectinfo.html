Project Overview:

This project is an interactive website designed to showcase the rich and diverse traditional clothing of India. The main feature of the website is a detailed map of India, with clickable buttons for each state. Each button directs users to a dedicated page highlighting the unique clothing designs and ethnic wear associated with that state.

Objectives:

*Cultural Education:

Provide an engaging platform for users to learn about the traditional attire and fashion of various Indian states.

*Interactive Experience:

Offer an interactive map that allows users to explore different regions and their unique clothing styles.

*Visual Appeal:

Use high-quality images and detailed descriptions to showcase the beauty and diversity of Indian ethnic wear.
